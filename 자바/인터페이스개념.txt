인터페이스
:  추상매서드 집합
:  구현된것이 없는 설계, 약속 (껍데기)
:  객체를 생성할 수 없다
:  자식이 상속을 통해서 구현한다   ( implements  키워드 사용)
:  상속을 통해서 완성해서 사용한다 ( 인터페이스 상속을 통해서 완성하는것을 구현이라고 한다)


  일반클래스 상속과  인터페이스 구현의 다른 점  ( 둘다 상속이라고 부른다)  부모로 사용될 수 있다 

:  직계의 물려줄게 있는 상속 : 일반상속
:  인터페이스 구현은  약속을 구현하는 것을 말한다

인터페이스 만들기
******

}

******
인터페이스 구현하기
******

class 클래스이름   implements 인터페이스이름{

}
